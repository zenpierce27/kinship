# Kinship

**Relationship Intelligence Platform**

An open-source relationship intelligence platform that goes beyond contact management. Kinship models the full complexity of human relationships — directionality, context, history, strength, and network effects — with AI-native semantic search and graph visualization.

## Core Features

- **Semantic Search** — Find contacts by meaning, not just keywords
- **Relationship Graph** — Visualize your network and connection paths
- **Warmth Tiers** — Track relationship strength (stranger → inner circle)
- **Interaction History** — Log calls, meetings, messages
- **Smart Reminders** — Never let important relationships go cold

## Architecture

- **Database**: Supabase (PostgreSQL + pgvector)
- **CLI**: TypeScript
- **Web**: Next.js
- **AI**: OpenAI embeddings for semantic search
